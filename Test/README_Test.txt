In this files are provided the commands that are needed to test the detector.
How to run:
    1- Unzip the folder content
    2- Open the terminal in the directory where the files have been extracted (test)
    3- Commands:
        a) mkdir build
        b) cd build
        c) cmake ..
        d) cmake --build .
        e) ./boat_detector --image=(Path to the image that must be tested) 
        
        ( example --image=/-------/image.jpg)
