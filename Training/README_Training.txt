In this files are provided the commands that are needed to test preparation.
How to run:
    1- Unzip the folder content
    2- Open the terminal in the directory extracted (dataset_preparation)
    3- Commands:
        a) mkdir build
        b) cd build
        c) cmake ..
        d) cmake --build .
        e) ./detector_training --folder_path=utility_folder contents path

            (the utility_folder is located in /----/PCV/Training/dataset_preparation/utility_folder then the path look similar to this)

The test is perfomed for only two images and the results can be seen in the directories boat and non_boat
